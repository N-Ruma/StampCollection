# Coding Rules
- **snake_case** for function declaration. e.g. "def map_view():", "def stamp_list_view():"
- **PascalCase** for class declaration.
- Use "POST" / "GET". **DO NOT USE "post" / "get"** in HTML requests.
- Use **snake_case** for all HTML file names and function declared in views.py, and end with "_view".  e.g. "map_view.html", "get_stamp_view.html" etc.

# コーディング規則
- 関数の宣言では，**スネークケース**を用いてください.
- クラスの宣言では，**パスカルケース**を用いてください. e.g. "class MapPin():"
- HTMLリクエストを指定する際, "POST" / "GET" を用いてください. "post" / "get" は使用しないでください.
- HTMLファイル名やviews.pyで宣言される関数は**スネークケース**を用いてください．また，これらの末尾は"_view"としてください. e.g. "map_view.html", "get_stamp_view.html" etc.
